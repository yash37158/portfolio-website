This Meme World App is purely build on Kotlin language which is Google's official language for Native App Development. Libraries being used are: Volley is an HTTP library that makes networking very easy and fast, for Android apps. Glide: Glide supports fetching, decoding, and displaying video stills, images, and animated GIFs. Glide includes a flexible API that allows developers to plug in to almost any network stack. Api reference: JSON API for a random meme scraped from reddit.
